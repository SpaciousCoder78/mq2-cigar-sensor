# mq2-cigar-sensor
An Arduino Sketch for MQ2 Smoke Sensor to detect Cigarette Smoke using ATmega328P boards

## Pre-Requisities
- Arduino IDE
- Arduino UNO
- MQ2 Smoke Sensor
- Wires
- Soldering Kit
